## 五、AI 服务流程

```
请求进入 AIService
    │
    ├── LLM_PROVIDER == "mock"?
    │   ├── 是 → MockLLM（正则/模板/规则引擎）→ 返回结果
    │   └── 否 → 加载 Prompt 模板
    │            → 调用 OpenAI 兼容 API（temperature=0.3）
    │            ├── 成功 → 解析结果
    │            │         ├── 解析成功 → 返回结果
    │            │         └── 解析失败 → 回退 MockLLM
    │            └── 调用失败 → 回退 MockLLM
```

**支持的 LLM Provider**：
- `mock`：内置 Mock，零成本
- `openai`：OpenAI API（GPT-4o-mini 等）
- `deepseek`：DeepSeek API（兼容 OpenAI 协议）
- `zhipu`：智谱 API（兼容 OpenAI 协议）

所有 Provider 共用同一套 Prompt 模板，通过 `OPENAI_BASE_URL` 和 `OPENAI_API_KEY` 切换。
