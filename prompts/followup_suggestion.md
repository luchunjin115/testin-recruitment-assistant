# 跟进建议生成 Prompt

## 对应模板

`backend/app/prompts/followup_suggestion.txt`

## 功能说明

根据候选人当前状态和背景信息，生成 1-2 句具体的跟进建议，帮助 HR 快速决定下一步操作。

## 设计要点

1. **行动导向**：建议必须包含"下一步应该做什么"，不是泛泛而谈
2. **紧急程度**：长时间未跟进的候选人自动提醒紧急度
3. **个性化**：结合候选人背景（学历、技能、岗位）给出针对性建议
4. **简洁输出**：直接输出建议文本，无标题无格式标记

## Mock 回退策略

`MockLLM.generate_followup()` 按招聘阶段匹配预设建议：

| 阶段 | 建议内容 |
|------|---------|
| 新投递 | 建议在24小时内完成初筛 |
| 初筛 | 尽快安排面试 |
| 待约面 | 联系候选人确定面试时间 |
| 已约面 | 提前准备面试评估表 |
| 面试中 | 及时收集面试官反馈 |
| 复试 | 综合评估，准备offer谈判 |
| offer | 跟进候选人入职意愿 |

**紧急追加**：若 `days_since_update > 3`，自动追加"该候选人已超过3天未跟进，建议优先处理"。

## 效果示例

输入：阶段=初筛、技能=Python/AI、3天未更新

输出：
> 该候选人具备AI方向技能，建议尽快安排技术面试以评估实际能力。注意：该候选人已超过3天未跟进，建议优先处理。
